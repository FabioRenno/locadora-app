# Cursor Context

Este projeto utiliza um arquivo de entrada para IA.

Antes de qualquer análise, sugestão de código ou refatoração, leia:

AI_ENTRYPOINT.md

Este arquivo contém:

- contexto do projeto
- stack atual
- fluxo principal do sistema
- prioridades do MVP
- regras de desenvolvimento

Após ler o AI_ENTRYPOINT.md, a IA deve respeitar as regras definidas em `.cursor/rules.md`.
